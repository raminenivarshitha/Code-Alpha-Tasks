# 🌍 AI Language Translator

A simple and interactive AI-powered Language Translator developed using **Python** and **Streamlit**. The application instantly translates text into **100+ languages**, provides **text-to-speech** for both original and translated text, maintains translation history, and offers a clean, responsive interface.

---

## 📌 Project Overview

The AI Language Translator is a user-friendly web application that enables users to translate text between multiple languages in real time. Built using **Streamlit**, **Google Translator**, and **gTTS**, it offers fast translations, speech playback, translation history, and downloadable results.

This project is ideal for students, professionals, travelers, and anyone who needs quick multilingual translations.

---

## ✨ Features

- 🌍 Translate text into 100+ languages
- 🤖 Automatic source language detection
- 🔄 One-click language swapping
- 🔊 Text-to-Speech for original text
- 🔊 Text-to-Speech for translated text
- 📜 Translation history with delete option
- 📊 Character counter
- 📥 Download translated text as a TXT file
- 📋 Sidebar showing translation history
- 🧹 Clear all translation history
- 🌙 Supports Light and Dark Mode
- ⚡ Fast and responsive Streamlit interface
- 📱 Clean and mobile-friendly UI

## 🚀 Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/AI-Language-Translator.git
```

### Step 2: Navigate to the Project Folder

```bash
cd AI-Language-Translator
```

### Step 3: Install Required Packages

```bash
pip install -r requirements.txt
```

### Step 4: Run the Application

```bash
streamlit run app.py
```

The application will automatically open in your default web browser.
