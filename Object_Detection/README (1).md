# 🔍 Smart Object Detection Dashboard

A web-based Object Detection application developed using **Python**, **Streamlit**, and **YOLOv8**. The application allows users to upload an image or capture one using a webcam, detects objects using the YOLOv8 model, and displays the results with bounding boxes and object statistics.

---

## 📌 Project Overview

The **Smart Object Detection Dashboard** is an AI-powered application that identifies and classifies objects in images using the YOLOv8 deep learning model. It provides an interactive interface where users can upload images or capture photos directly from their webcam for real-time object detection.

---

## ✨ Features

- 📤 Upload image for object detection
- 📷 Capture image using webcam
- 🎯 Detect multiple objects using YOLOv8
- 📦 Display bounding boxes around detected objects
- 📊 Detection summary with object count
- 🎚 Adjustable confidence threshold
- 💾 Download the detected image
- 📱 Responsive Streamlit dashboard
- 🌙 Clean and user-friendly interface

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Smart-Object-Detection.git
```

### Navigate to the project folder

```bash
cd Smart-Object-Detection
```

### Install the required libraries

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```

The application will automatically open in your default web browser.

---

## 📖 How to Use

1. Launch the application.
2. Choose a detection mode:
   - 📤 Upload Image
   - 📷 Webcam Capture
3. Adjust the confidence threshold using the slider.
4. Upload or capture an image.
5. View the detected objects with bounding boxes.
6. Check the detection summary.
7. Download the annotated image if needed.
