# 🤖 Tech Knowledge FAQ Chatbot

A simple and interactive chatbot developed using **Python** and **Streamlit** that answers frequently asked questions related to **Artificial Intelligence, Machine Learning, Cloud Computing, Python, SQL, Cybersecurity, Blockchain, and other Computer Science topics**.

---

## 📌 Project Overview

The Tech Knowledge FAQ Chatbot is a keyword-based chatbot designed to provide quick and accurate answers to commonly asked questions about trending technologies. It offers a clean chat interface, maintains conversation history, and serves as a beginner-friendly educational assistant.

---

## ✨ Features

- 💬 ChatGPT-style chat interface
- 🤖 Instant responses to technology-related questions
- 📚 Covers multiple Computer Science topics
- 📋 Sidebar displaying available topics
- 🗑 Clear Chat option
- 🌙 Supports both Light and Dark mode
- ⚡ Fast keyword-based response system
- 📱 Responsive and user-friendly interface

## 🚀 Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/Tech-FAQ-Chatbot.git
```

### Step 2: Navigate to the Project Folder

```bash
cd Tech-FAQ-Chatbot
```

### Step 3: Install Required Packages

```bash
pip install -r requirements.txt
```

### Step 4: Run the Application

```bash
streamlit run app.py
```

The chatbot will open automatically in your default web browser.
